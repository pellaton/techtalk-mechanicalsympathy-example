# techtalk-mechanicalsympathy-example

Examples of my TechTalk on mechanical sympathy 

## Example Code
### Prerequisites
* JDK 8 or 9
* Apache Maven

### Compile and Run JMH test
```
$ mvn clean package
$ java -jar target/benchmark.jar
```
